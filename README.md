# Computer Networks: Routing and Router Configuration Assignment

This repository contains the practical implementation of basic router configurations, manual path planning using Static Routing, and agile multi-node discovery using Dynamic Routing Protocols (RIPv2 and EIGRP) using **Cisco Packet Tracer**.

## 📌 Project Overview
* **Module:** Computer Networks
* **Submission Date:** June 15, 2026
* **Objective:** To configure routers, implement static routing, and configure dynamic routing protocols to establish seamless end-to-end connectivity across isolated local and wide area networks.
